# Persona Blog

Astro 4 · Tailwind CSS v4 · TypeScript strict · SSG (zero client JS)

## Key Paths

- `src/lib/` — shared utilities (mood engine, reading time, freshness logic)
- `src/data/` — config & content data (mood palette, /now page state)
- `src/components/` — reusable Astro components
- `src/layouts/` — page shells
- `src/pages/` — routes
- `src/content/` — blog collection (Markdown)
