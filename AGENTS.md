## Key Paths

- `src/lib/` — shared utilities
- `src/components/` — Astro components
- `src/layouts/` — page shells
- `src/data/` — JSON flat-file storage
- `src/styles/` — CSS modules
- `src/pages/api/` — SSR endpoints
- `cli/` — author CLI tools
